# Exercício 19 - Identificar erros (IF)

    - Identifique e corrija os erros em cada um dos itens seguintes. 

    - Obs: Comente o erro e a correção feita. 
        - a)
                se (idade >= 65); 
                    mostre ("melhor idade");

        - b)
                se (genero = 1) 
                    mostre ("Masculino")
                senão (genero == 2)
                    mostre ("Feminino"); 

        - c)
                se preco > 10.50
                    preco = preco * 1,2;
                senão
                    preco = preco * 1.35;
