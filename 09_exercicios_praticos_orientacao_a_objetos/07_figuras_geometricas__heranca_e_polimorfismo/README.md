# Exercício 07 - Figuras Geométricas (Herança e Polimorfismo)

    - Com o uso de herança e polimorfismo, crie um programa que:
        - Calcule a área de uma figura geométrica.

        - Aceite quatro tipos de figura geométrica:
            - Quadrado;
            - Retângulo;
            - Triângulo;
            - Círculo.
